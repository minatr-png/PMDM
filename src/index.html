AE3 asistente para recopilar datos de usuario

Esta aplicación es un asistente para la recopilación de datos que se divide en los siguientes pasos:
1- Un formulario a rellenar con los campos obligatorios señalados con un asterisco
2- Al completar el formulario se le enseñará los datos introducidos y deberá de confirmar si son correctos
3- En caso de que estos sean correctos se le redigirá a una página donde se verá que todo ha salido correctamente
Pero antes que nada debes introducir la callback URL para poder continuar
